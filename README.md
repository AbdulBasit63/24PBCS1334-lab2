# Lab 2 - Web Project

This repository contains the Lab 2 files for the Web Technologies course.

## Files:

- `index.html` - Home page
- `404.html` - Error page for non-existing URLs
- `navigation.html` - Navigation links
- `profile.html` - User profile page
- `registration.html` - Registration form
- `schedule.html` - Class schedule table
- `basit.jpg` - Sample image

## How to run

Open any `.html` file in a web browser to view the pages.
